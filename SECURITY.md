# Security Policy

## Supported Versions

The project is currently pre-release. Security fixes target the latest commit on the default branch.

## Reporting a Vulnerability

Please do not open a public issue for a suspected vulnerability.

Until a dedicated security contact is published, report vulnerabilities by opening a private GitHub security advisory for this repository. Include:

- Affected commit or version
- Steps to reproduce
- Impact
- Any relevant logs or screenshots

## Data Storage

TodoDesk stores issue data locally in the app's application data folder. Do not include private `issues.json` files in bug reports or pull requests.
